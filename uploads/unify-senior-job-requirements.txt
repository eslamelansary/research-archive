About the job
Job Title: Senior Node.js Developer 

Job Type: Full Time 

Job Location: Hybrid - Nasr City , Egypt 



We are seeking an experienced and highly skilled Senior Node.js Developer to join our team and contribute to the development of a groundbreaking IoT project built on a microservices architecture. As a Senior Node.js Developer, you will play a key role in designing and implementing scalable backend solutions using Node.js, as well as collaborating with cross-functional teams to deliver high-quality software solutions.


Responsibilities:

1. Backend Development:
 ◦ Design and develop scalable and efficient backend systems using Node.js, Express.js, and related technologies.
 ◦ Implement RESTful APIs and microservices to support the project's functionality and performance requirements.
 ◦ Collaborate with front-end developers, product managers, and other stakeholders to ensure seamless integration of backend services.
2. Architecture and Design:
 ◦ Architect and design reliable, scalable, and maintainable microservices using best practices and industry standards.
 ◦ Evaluate and select appropriate technologies, tools, and frameworks to optimize system performance and scalability.
 ◦ Collaborate with the team to define and implement coding standards, best practices, and design patterns.
3. Performance Optimization and Scalability:
 ◦ Identify and address performance bottlenecks and scalability challenges in the backend systems.
 ◦ Optimize code and database queries to improve system performance and response times.
 ◦ Implement caching strategies, load balancing techniques, and other performance optimization measures.
4. Testing and Quality Assurance:
 ◦ Develop and execute unit tests, integration tests, and end-to-end tests to ensure the reliability and stability of the backend systems.
 ◦ Collaborate with QA engineers to identify and resolve software defects and issues.
 ◦ Implement and maintain automated testing frameworks and tools to support continuous integration and deployment processes.
5. Documentation and Collaboration:
 ◦ Create and maintain technical documentation, including system architecture, API specifications, and code documentation.
 ◦ Collaborate effectively with cross-functional teams, including frontend developers, designers, product managers, and QA engineers.
 ◦ Communicate project status, technical challenges, and solutions to both technical and non-technical stakeholders.


Requirements:

• 4+ years of professional experience in backend development using Node.js and related technologies.
• Strong proficiency in Node.js, Express.js, and JavaScript.
• Solid understanding of microservices architecture, RESTful APIs, and distributed systems.
• Experience with database systems, both SQL and NoSQL.
• Proficient in writing efficient and scalable code, with a focus on performance optimization and scalability.
• Experience with containerization technologies like Docker and orchestration tools like Kubernetes.
• Strong knowledge of software development best practices, design patterns, and coding standards.
• Experience with testing frameworks and methodologies (e.g., Mocha, Chai, Jest).
• Excellent problem-solving skills and a strong attention to detail.
• Strong communication and collaboration skills, with the ability to work effectively in a team environment.
